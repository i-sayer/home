Scala files

From the Scala home page...

Scala is a powerful software tool for experimentation with musical tunings, such as just intonation scales, equal and historical temperaments, microtonal and macrotonal scales, and non-Western scales.

The scala application can retune midi instruments using pitch bender messages but for the G2, this will only work with monophonic patches.

While the G2 has no native support for scala files, a patch has been produced that allows for arbitrary returning of up to 96 notes within a polyphonic patch. The scala utility in this app can read scl files and directly adjust settings in the G2 scala patch template.

The scl file contains definitions for alternate tuning. For example

! meanquar.scl
!
1/4-comma meantone scale. Pietro Aaron's temperament (1523)

!

This app is intended as a replacement for the bare bones python version and will allow selecting up to eight scala files per patch.

Kbm (keyboard mapping) files are also supported, each scl file can have a kbm file associated with it. The app allows for the root key to be changed from the default Middle C, or the map file override. The selected note becomes the new root note (no tuning applied) while the octave is combined with any retuning specified in the map file.
